# Personal Newsfeed Assistant

This project aims to build an bot using Python that will show top news depending on user's category selection.

## About Newsbot:

Newsbot is a personal newsfeed bot that will show top 5 news based on user's category and country.

## Please see below for example on how this bot works:

1. Get the top news from different countries
	
	User: python3 newsbot.py general us
		
## Running on native machine
### Dependencies
* python3
* gtts

### using virtualenv (recommend)
1. `virtualenv newsbot.venv`
2. `source newsbot.venv/bin/activate`

### pip packages
`pip install -r requirements.txt` 

### In-Built libraries required to be imported:

	1.os
	2.requests
    3.playsound
	4.subprocess



Newsbot uses third party APIs to fetch news around the world.
